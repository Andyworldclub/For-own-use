插件 ：

（1）目前插件里面支持 ：[General],[rule],[rewrite],[host],[script],[mitm]

（2）插件是规则、复写、脚本的集合，相当于一个子配置，常常用来代表一个扩展功能。

（3）官方事例链接： https://nsloon.app/docs/Plugin/

使用方法：

操作流程：配置-插件-订阅插件

.lpx意为Loon Plugin Extension的缩写，即Loon插件扩展。
